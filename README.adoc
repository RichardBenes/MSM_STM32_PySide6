= MSM_STM32

== MEMS chips used

MEMS chips present on the board:

* https://www.st.com/en/mems-and-sensors/l3gd20.html[L3GD20] -- 3-axis gyroscope
* https://www.st.com/en/mems-and-sensors/lsm303c.html[LSM303C] -- compass + 3D accelerometer + 3D magnetometer
* https://www.st.com/en/audio-ics/mp34dt01-m.html[MP34DT01-M] -- MEMS microphone
